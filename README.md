# Robot_frameworkdemo

Robot Framework-google search scrape

       1.In this project, https://www.google.co.in has been automated with the search text "robotframework"
       2.And the results/links of the page has printed in the console and also the results has been pushed to text file name "Textfile.txt".
       3.Automated screenshot has been took automatically to verify and pushed to the directory with file named "Automatic_screenshot"
       4.Finally the manual screenshot of the console has been pushed to the "Screenshot" folder.
